# SPEC — Focus Sprint

## App description
Focus Sprint is a tiny CLI app that creates structured 25/5 focus sessions and logs outcomes to a local JSON file.

## Target user
Solo builders and students who want a no-friction Pomodoro-style workflow directly in terminal.

## MVP
- Start a focus session with configurable duration (default 25 minutes).
- End session and record whether completed/abandoned.
- Show a daily summary of completed sessions.
- Store data locally in `data/sessions.json`.
- Provide clear CLI help and error handling.
